# Survey Data Analysis & Quality Validation

##  Project Overview
This project focuses on analyzing survey data such as data validation and defensive programming to handle missing values and inconsistent entries.

##  Tools Used
- **Python** (Pandas, NumPy)
- **Visualization**: Seaborn, Matplotlib
- **Validation**: Custom error-handling for empty data series.

##  Key Insights
- **Robust Analysis**: Used `.iloc` and `.empty` checks to prevent crashes during data aggregation.
- **Visual Trends**: Identified the most frequent responses across multiple survey categories.

##  How to Run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook in `/notebooks`.
